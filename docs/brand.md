# Identité FootLumen

FootLumen est le nom public de l'application.

Tagline :

```text
Éclaire le match par la data.
```

## Positionnement

FootLumen met en avant la clarté, la donnée, l'analyse football et la prudence
probabiliste. L'identité ne doit pas utiliser d'imagerie liée aux paris, à l'argent,
aux gains, au casino, aux tickets ou aux jetons.

## Palette

- Bleu profond : `#123B73`
- Cyan lumineux : `#22C7E8`
- Vert terrain : `#27C77A`
- Blanc : `#FFFFFF`

## Assets

Le pack source est versionné dans :

```text
footlumen_identity_pack/
```

Contenu principal :

- logos SVG modifiables ;
- exports PNG ;
- favicon et icône d'application ;
- bannières Discord ;
- mini-charte HTML ;
- planche de prévisualisation.

## Compatibilité Technique

La commande CLI reste `football-predictor` et le package Python reste
`football_predictor`. Ces noms sont techniques et conservés pour éviter de casser les
crons, imports, tests, scripts VPS et installations existantes.
